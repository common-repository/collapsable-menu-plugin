== Description ==

This plugin is a collapsable menu plugin. I have just created a collapsable menu plugin. This plugin is very useful if you have child category. For example if you have finance category, stock and option subcategory. You would only create article and sign it to the subcategory. You can comment <a href="http://smallwebsitehost.com/collapsable-menu-plugin-for-wordpress/wordpress/" target="_blank">here</a>.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php wp_list_categoriesex(); ?>` in your templates, posibly sidebar.php.
